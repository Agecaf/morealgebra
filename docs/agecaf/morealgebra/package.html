    Contains Constructors and Syntax for groups, and some Standard Groups.

    Contains Constructors and Syntax for groups, and some Standard Groups.

    This package contains an ample set of constructor methods for groups, as well as some basic groups. It additionally includes the group syntax from Spire, so that further imports are not required.

    Implicits

    • Spire's syntax implicit conversions acting on group members. This adds the |+| operator, for example.
    • Spire's syntax implicit conversions for Eq. This adds the === operator, for example. Used for Cyclic groups for example.
    • MoreAlgebra's group implicit conversions. These are for syntax sugar for DirectProducts for example. Note that these act on groups Group[A] rather than on group members A

    Non Implicits

    • No instances of Group[A] are given implicitly. This is a design choice.
    • No instances of Eq[A] are given implicitly. This is a design choice.
